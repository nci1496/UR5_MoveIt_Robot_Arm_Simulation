# 操作文档

本文档为视频演示的操作指南，包含完整的运行命令和操作步骤。

---

## 准备工作

**录制前检查**：
- [ ] Ubuntu 20.04 系统正常运行
- [ ] ROS Noetic 已安装（`rosversion -d` 验证）
- [ ] UR5仿真包已安装
- [ ] 代码已编译（`catkin_make`）

---

## 演示顺序

建议按以下顺序演示，每个功能一边讲解一边运行：

1. 环境启动（1分钟）
2. 六点点位控制（2分钟）
3. 笛卡尔直线轨迹（1分钟）
4. 椭圆轨迹（2分钟）
5. 8字形轨迹（2分钟）
6. 虚拟画板系统（3分钟）
7. 总结（1分钟）

---

## 第1步：启动仿真环境

**操作**：打开终端1，输入命令

```bash
roslaunch ur5_moveit_config demo.launch
```

**讲解内容**：
> 这是启动MoveIt仿真环境的命令。执行后，RViz会加载UR5机械臂模型。

**等待**：等待RViz完全启动，机械臂模型显示正常（约10秒）

**截图**：截取RViz界面作为图2-1

---

## 第2步：六点点位控制

**操作**：打开终端2，输入命令

```bash
rosrun arm_control_demo six_points
```

**讲解内容**：
> 六点点位控制是最基础的运动方式。机械臂会依次到达六个预定义的空间点位。
> 终端会显示"Move to point X Success"，表示每个点位都成功到达。

**观察**：机械臂在RViz中依次移动到6个位置

**截图**：截取终端输出+RViz作为图3-1

---

## 第3步：笛卡尔直线轨迹

**操作**：终端2，输入命令

```bash
rosrun arm_control_demo cartesian_line
```

**讲解内容**：
> 笛卡尔直线轨迹使用MoveIt的computeCartesianPath函数。
> 机械臂末端会沿直线路径平滑移动，避免奇异构型。

**截图**：截取RViz作为图3-2

---

## 第4步：椭圆轨迹

**操作**：终端2，输入命令

```bash
rosrun arm_control_demo ellipse_path
```

**讲解内容**：
> 椭圆轨迹通过数学方程生成。
> 参数方程：x = 0.4 * cos(t), y = 0.3 * sin(t)
> 红色Marker显示的是规划路径，机械臂沿此路径运动。

**观察**：
- 终端显示 "Fraction = 1.00"（100%规划成功）
- 红色Marker显示椭圆曲线
- 机械臂末端绘制椭圆

**截图**：
- 截取Marker可视化作为图3-3
- 截取执行过程作为图3-4

---

## 第5步：8字形轨迹

**操作**：终端2，输入命令

```bash
rosrun arm_control_demo eight_path
```

**讲解内容**：
> 8字轨迹是两个椭圆在垂直方向的叠加。
> 关键在于Y坐标使用 sin(2t)，产生两倍频率。
> 这展示了复杂三维轨迹的控制能力。

**观察**：
- 红色Marker显示8字形状
- 机械臂末端沿8字路径运动

**截图**：截取Marker可视化作为图3-5

## 第6步：虚拟画板系统

### 6.1 启动所有组件

**操作**：

```bash
# 终端1（如果已关闭，重新启动RViz）
roslaunch ur5_moveit_config demo.launch

# 终端2：启动Qt GUI
rosrun virtual_canvas_gui virtual_canvas_gui

# 终端3：启动执行器
rosrun canvas_executor canvas_executor
```

**讲解内容**：
> 虚拟画板系统由三部分组成：
> 1. RViz仿真环境（终端1）
> 2. Qt画板GUI（终端2）- 鼠标绘图
> 3. 执行器节点（终端3）- 控制机械臂

**等待**：所有窗口启动完成（约5秒）

### 6.2 GUI功能介绍

**操作**：展示Qt GUI界面

**讲解内容**：
> Qt界面包含：
> - 左侧：绘图区域，鼠标自由绘图
> - 右侧：控制面板
>   - 速度滑块：1%-100%调节
>   - 采样率：调节轨迹精度
>   - 镜像控制：X轴/Z轴镜像
>   - 执行按钮：发送轨迹
>   - 清空按钮：清除画布

**截图**：截取GUI界面作为图4-1

### 6.3 演示绘图和执行

**操作**：
1. 在GUI中用鼠标画一个简单图形（如字母L）
2. 点击"执行"按钮

**讲解内容**：
> 现在我画了一个简单的L形。
> 点击"执行"后，轨迹会通过ROS话题发布。
> 执行器接收后控制机械臂复现绘制。

**观察**：
- 终端3显示 "Trajectory published to /canvas_trajectory"
- RViz中显示红色Marker轨迹预览
- 机械臂开始运动

**截图**：
- 截取GUI绘图状态作为图4-2
- 截取RViz执行作为图4-3
- 截取最终效果作为图4-4
